Pokemon Mystery Dungeon Autonomous Agent Demo

This demo showcases an AI agent playing Pokemon Mystery Dungeon Red Rescue Team using Qwen3-VL vision models. The agent navigates Tiny Woods B1F, makes decisions, and solves the floor autonomously.

Demo Details:

  • Agent uses multi-model Qwen3-VL (2B/4B/8B variants)
  • Hierarchical RAG with 7 temporal resolution silos
  • Dynamic FPS adjustment for optimal performance
  • Live dashboard integration (when available)
  • Skill-based action execution

For more information, see the GitHub repository.
